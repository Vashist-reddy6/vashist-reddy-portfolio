# Vashist Reddy — Portfolio

Personal portfolio for Vashist Reddy Mosra, built with Next.js and designed around an editorial, content-first visual system.

## What it includes

- Selected work: ScrapeIQ and WeatherGuard AI
- Education, technical focus, achievements, and certifications
- GitHub, LinkedIn, email, and downloadable résumé links
- Responsive layout, keyboard navigation, and reduced-motion support

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run lint
npm run build
```

The repository is ready for automatic deployment on Vercel.
